AIDORag Widget Test Page

This is a test page for the AIDORag embeddable chat widget. Look at the bottom-right corner for the chat button.

How to Test:

  1. Create a widget in the Admin Panel
  2. Copy the widget key (e.g., wgt_abc123...)
  3. Replace YOUR_WIDGET_KEY below with your key
  4. Click the chat button to start a conversation
<script src="http://localhost:3000/widget/aidorag-widget.js" data-widget-key="YOUR_WIDGET_KEY"> </script>
Note: Replace the widget key in this file or pass it via URL parameter:
test.html?key=wgt_xxx
Go to Widget Admin
